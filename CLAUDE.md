# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

Govee smart lamp integration that reacts to Claude Code's states via hooks. The lamp changes color:
- Cool blue/teal (0, 180, 220) — Idle
- Animated scene (or amber fallback 255, 160, 40) — Working
- Vivid red (255, 30, 30) — Input required

## Key Files

- `govee_status.py` — Main script. Called with one arg: `idle`, `working`, `input_required`, or `--discover-scenes`. Sends color/scene to Govee API with 2-second debounce.
- `setup_device.py` — One-time device discovery. Lists Govee devices, user picks one, writes `config.json` (including available dynamic scenes).
- `config.json` — Device ID, SKU, API key, cached scenes. Gitignored. Generated by `setup_device.py`.

## Govee API

- Base URL: `https://openapi.api.govee.com`
- Auth: `Govee-API-Key` header
- Control: `POST /router/api/v1/device/control`
- Color value: single integer `(R << 16) | (G << 8) | B`
