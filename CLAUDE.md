# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

Govee smart lamp integration that reacts to Claude Code's states via hooks. Two states:
- Warm white (255, 220, 200) — Claude is working or session just started
- Red (255, 0, 0) — Waiting for user (idle, permission needed, or clarifying question)

## Key Files

- `govee_status.py` — Main script. Called with one arg: `idle`, `working`, or `input_required`. Sends color to Govee API with 2-second debounce. Uses fire-and-forget subprocess for non-blocking hook return.
- `setup_device.py` — One-time device discovery. Lists Govee devices, user picks one, writes `config.json`.
- `config.json` — Device ID, SKU, API key. Gitignored. Generated by `setup_device.py`.

## Govee API

- Base URL: `https://openapi.api.govee.com`
- Auth: `Govee-API-Key` header
- Control: `POST /router/api/v1/device/control`
- Color value: single integer `(R << 16) | (G << 8) | B`
